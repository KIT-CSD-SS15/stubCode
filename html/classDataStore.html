Detailed Description

template<class Storable>
class DataStore< Storable >

A Singleton for communication of objects across modules.

Use the Instance function to access the Singleton. For each class of objects communicated a separate DataStore will be instantiated. The DataStore consists basically of a map with std::string as key and one of the transfered objects as mapped.
