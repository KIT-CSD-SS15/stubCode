void Chamber::cleanUp ( )

"Empties" all cells by resetting their energy deposition.

This function is used as well in the GeometryCreator.

ChamberIterator Chamber::first ( ) const

Returns an iterator connected to the first Cell.

You can use this iterator to loop over all Cells of the Chamber. There is no end() function implemented, but an interator will return a nullptr, if it is iterated over the last element.
